# 🥗 NutriTrack

Plataforma digital para la gestión y seguimiento nutricional que conecta a nutriólogos con pacientes mediante una solución web y móvil.

---

## 📖 Descripción del Proyecto

NutriTrack es una plataforma dual diseñada para digitalizar y profesionalizar la relación entre nutriólogos y pacientes, centralizando información clínica, planes alimenticios y métricas de progreso en un solo sistema.

El objetivo es mejorar la adherencia al tratamiento nutricional mediante seguimiento estructurado, comunicación directa y visualización de resultados.

---

## 🚩 Problema que Resuelve

Actualmente muchos nutriólogos gestionan información en:

- WhatsApp  
- Excel  
- Correos electrónicos  

Esto provoca:
- Desorganización del historial clínico  
- Pérdida de seguimiento  
- Abandono del tratamiento  
- Falta de profesionalismo  

NutriTrack centraliza toda la información en una sola plataforma.

---

## 🎯 Objetivos del Proyecto

- Digitalizar el seguimiento nutricional.
- Permitir creación y asignación de dietas.
- Registrar métricas (peso, grasa corporal, IMC).
- Generar gráficas de progreso.
- Facilitar comunicación directa entre paciente y nutriólogo.

---

## 🏗 Alcance (MVP)

### 👩‍⚕️ Para Nutriólogo (Web)
- Registro e inicio de sesión
- Gestión de pacientes
- Creación y edición de dietas
- Registro y consulta de métricas
- Revisión de progreso
- Ajuste de dieta

### 📱 Para Paciente (App)
- Registro e inicio de sesión
- Visualización de menú diario
- Registro de peso
- Visualización de progreso
- Chat con especialista

---

## 🔄 Proceso de Negocio

El sistema funciona bajo un modelo de seguimiento continuo:

1. Nutriólogo asigna dieta.
2. Paciente registra métricas periódicamente.
3. Sistema actualiza historial y genera gráficas.
4. Nutriólogo evalúa progreso.
5. Se ajusta dieta si es necesario.
6. El ciclo continúa hasta cumplir objetivos.
